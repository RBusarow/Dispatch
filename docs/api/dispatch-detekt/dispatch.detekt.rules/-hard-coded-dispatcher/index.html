dispatch-detekt/dispatch.detekt.rules/HardCodedDispatcher

HardCodedDispatcher

Detects use of a hard-coded reference to the kotlinx.coroutines.Dispatchers singleton.

The CoroutineDispatcher's in this singleton do not contain a DispatcherProvider, so they're unaffected by this library.

Config

  • allowDefaultDispatcher

  • allowIODispatcher

  • allowMainDispatcher

  • allowMainImmediateDispatcher

  • allowUnconfinedDispatcher
